# Oasis Restaurant - QR Menu

Restoran va kafeler uchun QR kod orqali ochiladigan mobil menyu sayti.

## Xususiyatlari

- Telefon ekraniga to'liq moslashgan (responsive) dizayn
- Kategoriyalar bo'yicha filtr
- Taom qidirish funksiyasi
- Savat tizimi
- Telegram orqali buyurtma yuborish
- Splash screen animatsiya
- Tez va yengil — faqat HTML, CSS, JavaScript

## Ishga tushirish

`index.html` faylini brauzerda oching yoki istalgan web serverda joylashtiring.

## Texnologiyalar

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display, Inter, Dancing Script)
